[RUS] 

Transaction Checker bot - бот для отслеживания последних транзакций пользователя по его токену https://bscscan.com/. 

Для вызова команды просмотра транзакций напишите “/TC” или “/tc”. Дальше через пробел укажите токен человека. После через ещё один пробел можно указать количество последних транзакций, которые хотите увидеть. По умолчанию эта опция имеет значение 5. 

[ENG] 

Transaction Checker bot - a bot for tracking a user's latest transactions using his token https://bscscan.com/.  

To call the transaction checker bot, type "/TC" or "/tc". Next, after a space, write the person's token. After another space, you can specify the number of recent transactions you want to see. The default value of this option is 5.
